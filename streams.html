filter Lazy
Keeps elements that match the predicate.
stream.filter(s -> s.length() > 3)
distinct Stateful
Removes duplicates based on equals().
stream.distinct()
limit Short-circuit
Truncates the stream to allow no more than 'n' elements.
stream.limit(5)
skip Lazy
Discards the first 'n' elements.
stream.skip(2)
takeWhile Java 9
Takes elements while condition is true, then stops.
stream.takeWhile(n -> n < 10)
dropWhile Java 9
Drops elements while condition is true, takes rest.
stream.dropWhile(n -> n < 10)
üîÑ Mapping & Transforming
map Lazy
Applies a function to each element (1-to-1).
stream.map(String::toUpperCase)
mapToInt Primitive
Unboxes to IntStream (also mapToLong, mapToDouble).
stream.mapToInt(User::getAge)
flatMap Lazy
Flattens nested structures (1-to-Many).
stream.flatMap(List::stream)
flatMapToInt Primitive
Flattens to IntStream (also flatMapToLong, etc).
stream.flatMapToInt(x -> IntStream.of(x))
üìä Sorting & Debugging
sorted Stateful
Sorts in natural order. Warning: Expensive on large streams.
stream.sorted()
sorted(Comp) Stateful
Sorts using a custom Comparator.
stream.sorted(Comparator.reverseOrder())
peek Debug
Performs an action without modifying the stream.
stream.peek(System.out::println)
unordered Optimize
Relaxes ordering constraint for faster parallel processing.
stream.unordered()
parallel Mode
Switches the stream to parallel execution mode.
stream.parallel()
onClose Hook
Adds a handler to run when close() is called.
stream.onClose(() -> log("Done"))
üîç Finding & Matching
findFirst Optional
Returns the first element. Deterministic.
stream.findFirst()
findAny Optional
Returns any element. Faster in parallel.
stream.findAny()
anyMatch Boolean
Returns true if at least one element matches.
stream.anyMatch(s -> s.isEmpty())
allMatch Boolean
Returns true if all elements match.
stream.allMatch(n -> n > 0)
noneMatch Boolean
Returns true if no elements match.
stream.noneMatch(s -> s.contains("fail"))
üì¶ Reducing & Collecting
collect Mutable
Accumulate into List, Map, Set, or custom container.
stream.collect(Collectors.toList())
reduce Immutable
Combine elements into a single value (3 variations).
stream.reduce(0, (a, b) -> a + b)
min / max Optional
Finds smallest or largest element using Comparator.
stream.min(Integer::compare)
count Long
Returns the count of elements.
stream.count()
toArray Array
Converts stream elements to an array.
stream.toArray(String[]::new)
üîÑ Iterating
forEach Void
Runs an action for each element (Side-effects).
stream.forEach(System.out::println)
forEachOrdered Void
Runs action strictly maintaining encounter order.
stream.forEachOrdered(printFn)
